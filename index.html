Panduan Android

  1. Unduh PowerTunnel
  2. Klik PowerTunnel URL di atas
  3. Buka PowerTunnel → Host Settings
  4. Tempel URL, atur update on start
  5. Buka Growtopia!
phone_iphone

Panduan iOS

  1. Unduh Surge 5
  2. Import → Download Profile from URL
  3. Masukkan URL dari iOS Host URL
  4. Setup VPN, buka Growtopia
laptop_windows

Panduan Windows

  1. Win+R, ketik C:\Windows\System32\drivers\etc
  2. Buka hosts dengan Notepad (Admin)
  3. Tambahkan
  4. Simpan, buka Growtopia
